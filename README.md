e-commerce mysql analysis
E-commerce MySQL Analysis project using a 3-table dataset (Orders, Order Details, Category Targets). Includes data cleaning, EDA, joins, sales trends, category performance, profitability insights, and target vs actual analysis using SQL. Perfect for analytics practice.


ðŸ”¹ Dataset Overview

The dataset contains three relational tables:

1. Orders

Customer-level order information.
Columns: Order ID, Order Date, Customer Name, State, City

2. Order Details

Line-level sales details.
Columns: Order ID, Amount, Profit, Quantity, Category, Sub-Category

3. Category Target

Target performance for each product category by month.
Columns: Month, Category, Target

Data source : https://www.kaggle.com/datasets/benroshan/ecommerce-data/
